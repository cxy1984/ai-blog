通过 Telegram Bot API 发送消息给个人或群组

📋 使用说明

  1. 创建 Telegram 机器人：在 Telegram 中搜索 @BotFather，发送 /newbot 命令创建新机器人
  2. 获取 Token：创建成功后，BotFather 会提供机器人的 API Token，格式如 123456789:ABCdefGHIjklMNOpqrsTUVwxyz
  3. 获取 Chat ID：
    • 私聊：发送消息给机器人后，访问 https://api.telegram.org/bot<这里填入你的完整Token>/getUpdates
    • 群聊：将机器人拉入群组，发送消息后访问同样的 URL 获取 Chat ID
  4. API 方法说明：
    • GET 请求：在浏览器中直接访问上述 URL，返回 JSON 格式的更新数据
    • POST 请求：通过编程方式调用，需要在请求体中发送 JSON 数据
    • 本工具使用 POST：通过表单提交，自动构建 POST 请求发送到 sendMessage 端点
  5. 获取步骤详解：
    • 首先确保已向机器人发送过至少一条消息（私聊或群聊都可以）
    • 将你的完整 Token 替换到 https://api.telegram.org/botYOUR_TOKEN_HERE/getUpdates 中的 YOUR_TOKEN_HERE
    • 在浏览器中访问该完整URL，复制返回结果中的 chat.id 数值
    • 个人聊天 ID 为正数，群组聊天 ID 为负数（以 - 开头）
  6. 填写参数：在下方表单中填入获取的 Token 和 Chat ID
  7. 编写消息：在消息框中输入要发送的内容
  8. 发送消息：点击发送按钮完成消息发送
