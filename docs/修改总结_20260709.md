## 4. 日期显示逻辑优化

### 修改原因

原脚本使用文件修改时间（mtime）作为文章日期。编辑文件后日期会错误地变为编辑当天。

### 修改文件

| 文件 | 改动 |
|------|------|
| `src/generate_index_with_dates.js` | 重写 `getFileDate()` 为 3 级优先级 |
| `src/generate_overview.js` | 同上 |

### 日期优先级（从高到低）

```
优先级 ①  文章内容中 > 日期：YYYY-MM-DD
         ↓ 没有则
优先级 ②  文件修改时间（mtime）
         ↓ 没有则
优先级 ③  文件夹名（如 2026-06 → 当月1号）
```

### 补充日期标记

`markdown/2026-06/这个博客项目是怎么运行和发布的.md` 新增 `> 日期：2026-06-12`
