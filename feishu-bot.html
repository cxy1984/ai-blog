📋 飞书机器人

通过飞书机器人 API 发送消息到群聊或个人

← 返回工具中心

📋 使用说明

  1. 创建飞书机器人：
    • 打开飞书电脑版或网页版
    • 进入群聊设置 → 群机器人 → 添加机器人 → 自定义机器人
    • 设置机器人名称和描述，获取 Webhook URL
  2. 获取 Webhook URL：创建机器人后会获得形如 https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx 的Webhook地址
  3. 选择消息类型：支持文本消息和富文本消息两种格式
  4. 编写消息内容：根据选择的消息类型编写对应的内容
  5. 发送消息：点击发送按钮推送到飞书群聊或用户
从飞书群聊机器人设置中获取的 Webhook 地址
纯文本消息，最长不超过 2000 字符
支持 Markdown 格式，如 **粗体**, *斜体*, `代码`
